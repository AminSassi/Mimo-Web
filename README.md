# MiMo Auto v2.2.0

One-click installer for MiMo Code — an AI coding assistant.

## Download

Download **MiMoSetup.exe** from this page. Right-click → Run as Administrator.

## What it installs

- Node.js v20.15.1 (portable)
- Git MinGit v2.45.2 (portable)
- MiMo CLI v0.1.0

## Quick start

1. Download `MiMoSetup.exe`
2. Right-click → Run as administrator
3. Follow the installer
4. Double-click **MiMo Web** on your Desktop

## Need help?

- Run `factory_reset.cmd` as admin, reboot, then try again
- Check logs at `C:\Program Files\MiMo Auto\logs\bootstrapper.log`

## For developers

Source code and build scripts are in the `Content/` folder.
